# Car Price Prediction

A project that predicts used car prices based on various features such as manufacturer, model, production year, mileage, fuel type, and more. The prediction model uses machine learning with preprocessing including one-hot encoding, label encoding, and feature scaling.

---

## Features

- Predict car prices using a trained ML model.
- Input validation with Pydantic.
- One-hot and label encoding for categorical features.
- Feature scaling for numerical columns.
- Clean and production-ready FastAPI API.
